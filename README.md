# BenefitIQ - American Express CodeStreet 2026

ML-driven benefit underutilization analytics engine.

## Problem
$2-3B lost to unused benefits annually.

## Solution
XGBoost model predicting benefit utilization + personalized nudges.

## Tech Stack
- Backend: Flask
- Frontend: Streamlit
- ML: XGBoost
- Data: Kaggle + 50K synthetic users

## Setup
pip install -r requirements.txt
